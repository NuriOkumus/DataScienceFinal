# Data Science Study Repository

This repository serves as a comprehensive collection of resources for learning and practicing various data science concepts using R. It includes theoretical notes, practical code implementations, self-assessment materials, and a Next.js-based website to present the content.

## Project Structure

```text
.
├── Codes/                      # R scripts for practical applications and data analysis
│   ├── 01_Mice_Uygulama.R      # Application of MICE (Multivariate Imputation by Chained Equations) for missing data
│   ├── 02_PCA_Uygulama.R       # Application of Principal Component Analysis (PCA)
│   ├── 03_Gorsellestirme.R     # Data visualization examples and techniques
│   ├── 04_Regresyon.R          # Regression analysis implementations
│   └── 05_Siniflandirma.R      # Classification algorithms implementations
├── Notes/                      # Theoretical notes corresponding to the code topics
│   ├── 01_Eksik_Veri_Teorisi.md # Theory behind missing data handling
│   ├── 02_PCA_Teorisi.md       # Theory of Principal Component Analysis
│   ├── 03_Gorsellestirme.md    # Principles of data visualization
│   ├── 04_Regresyon_Varsayimlari.md # Assumptions underlying regression analysis
│   └── 05_Siniflandirma_Teorisi.md # Theory of classification models
├── TestYourSelf/               # Resources for self-assessment and practice
│   └── Final_Sorulari_Cozumleri.md # Solutions to final exam/practice questions
├── WebSite/                    # Source code for the project's website (Next.js)
│   ├── components/             # React components for the website
│   ├── pages/                  # Next.js pages and routing
│   ├── public/                 # Static assets (images, etc.)
│   ├── types/                  # TypeScript type definitions
│   └── theme.config.tsx        # Theme configuration (likely for a documentation site)
├── final_calisma_plani.md      # A structured study plan
└── test_yourself_questions.md  # A collection of questions to test your understanding
```
